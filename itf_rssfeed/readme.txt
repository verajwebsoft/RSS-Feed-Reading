This plug-in will scroll the RSS feed title vertically in the widget, admin can add/update the RSS link & style via widget management.

Vertically scroll rss feed wordpress plugin will create the scroll news in the website widget. We can provide the rss feed as a input for this scroll. That is, This plugin read the given rss feed and create the vertical scroll with rss feed title. We can use only one rss feed at a time.

**Features of this plugine** 

* Easy installation.
* Can add any RSS feed.
* Easy style-override system.
* Scrolling, So occupy less space.

What is rss? RSS (Really Simple Syndication) is a family of web feed formats used to publish frequently updated worksâ€”such as blog entries, news headlines.

**Plugin configuration**

* Drag and drop the widget.

* Add directly in the theme.
